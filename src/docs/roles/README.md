# Role-Specific OpenAPI Specs

Generated by `npm run build:openapi`. Each file contains only the endpoints allowed for that role.

- `swagger-role-admin.json` - Admin APIs
- `swagger-role-gm.json` - General Manager APIs
- `swagger-role-tm.json` - Technical Manager APIs
- `swagger-role-surveyor.json` - Surveyor APIs
- `swagger-role-client.json` - Client APIs

Access at:
- /api-docs/admin
- /api-docs/gm
- /api-docs/tm
- /api-docs/surveyor
- /api-docs/client
